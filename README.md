# Internship Application Tracker

A full-stack web app that connects to your Gmail and automatically categorizes your internship applications — no manual logging required.

## Features

- **Gmail OAuth** — sign in with Google, read-only access to your inbox
- **Auto-classification** — emails sorted into No Response, Rejected, or OA
- **Dashboard** — summary cards, application funnel chart, and filterable email table
- **No data stored** — emails are fetched live on each session, nothing is saved server-side

## Tech Stack

- **Backend** — Python, FastAPI, Gmail API, JWT auth, deployed on Railway
- **Frontend** — React, Vite, Nivo charts, deployed on Vercel

## Usage

Visit the live app, sign in with Google, and your inbox is scanned automatically.
